Introduction to Kokkos

Kokkos is a library inside the Trilinos that "implements a programming model in C++ for writing performance portable applications targeting all major High Performance Computing(HPC) platforms". Therefore, Kokkos "provides abstractions for both parallel execution of code and data management".

Learning Kokkos
